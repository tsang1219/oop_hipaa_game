## Spritesheet Format

| Property        | Value                                          |
|----------------|------------------------------------------------|
| File size       | 96 × 128 pixels per sheet                     |
| Frame size      | 32 × 32 pixels per frame                      |
| Grid layout     | 3 columns × 4 rows                             |
| Transparency    | RGBA (transparent background)                  |

### Row Order (direction)

| Row | Direction     | Camera relative                  |
|-----|--------------|----------------------------------|
| 0   | Walk Down    | Facing toward camera (front view)|
| 1   | Walk Left    | Profile — facing left            |
| 2   | Walk Right   | Profile — facing right           |
| 3   | Walk Up      | Facing away from camera (back)   |

### Column Order (animation frame)

| Column | Frame       | Description                             |
|--------|------------|----------------------------------------|
| 0      | Idle        | Standing still, neutral pose           |
| 1      | Walk A      | Step — one foot forward                |
| 2      | Walk B      | Step — opposite foot forward           |

**Walk animation loop:** Frame 0 (idle) → Frame 1 (walk-A) → Frame 0 (idle) → Frame 2 (walk-B) → repeat

### Phaser 3 Load Configuration

```typescript
// Load a single character sheet
this.load.spritesheet('player', '/attached_assets/generated_images/privacyquest/characters/player.png', {
  frameWidth: 32,
  frameHeight: 32
});
```

**Frame index formula:**
- `row * 3 + col` = frame index
- e.g., Walk Down Idle = 0×3+0 = frame **0**
- e.g., Walk Down Walk-A = 0×3+1 = frame **1**
- e.g., Walk Left Idle = 1×3+0 = frame **3**
- e.g., Walk Right Walk-B = 2×3+2 = frame **8**
- e.g., Walk Up Idle = 3×3+0 = frame **9**
