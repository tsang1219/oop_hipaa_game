**Tile color logic:**
- `completed` (in completedRooms): bg `#1a4a1a` border `#44ff44` text `#44ff44`
- `current` (is currentRoomId): bg `#3a3000` border `#ffd700` text `#ffd700` with CSS pulse animation
- `locked` (not in unlockedRooms): bg `#1a1a1a` border `#333333` text `#555555` opacity-60
- `available` (in unlockedRooms, not current, not completed): bg `#1a1a2e` border `#4ECDC4` text `#ffffff`

**Pulse animation for current tile:** Use Tailwind `animate-pulse` class on the border element, or a CSS keyframe via `style` prop. Keep it subtle — a 2s ease-in-out opacity pulse on the border color.

**Layout:** Horizontal strip, positioned `absolute bottom-2 left-1/2 -translate-x-1/2` (centered bottom of canvas div). Use `pointer-events-none` so it doesn't intercept clicks. `z-[10]`.

```tsx
export function DepartmentBreadcrumb({
  completedRooms, currentRoomId, currentAct, unlockedRooms
}: DepartmentBreadcrumbProps) {
  return (
    <div
      className="absolute bottom-2 left-1/2 -translate-x-1/2 flex items-center gap-1 bg-black/70 border border-[#333] px-2 py-1 pointer-events-none"
      style={{ fontFamily: '"Press Start 2P"', fontSize: '6px', zIndex: 10 }}
      data-testid="department-breadcrumb"
    >
      {/* Act label */}
      <span className="text-[#ffd700] mr-2 text-[6px]">ACT {currentAct}</span>

      {/* Department tiles */}
      {DEPARTMENT_ORDER.map(dep => {
        const isCompleted = completedRooms.includes(dep.id);
        const isCurrent = dep.id === currentRoomId;
        const isLocked = !unlockedRooms.includes(dep.id);

        // Determine tile state
        let bgColor = '#1a1a2e';
        let borderColor = '#4ECDC4';
        let textColor = '#ffffff';
        if (isCompleted) { bgColor = '#1a4a1a'; borderColor = '#44ff44'; textColor = '#44ff44'; }
        else if (isCurrent) { bgColor = '#3a3000'; borderColor = '#ffd700'; textColor = '#ffd700'; }
        else if (isLocked) { bgColor = '#1a1a1a'; borderColor = '#333333'; textColor = '#555555'; }

        return (
          <div
            key={dep.id}
            title={dep.fullName}
            className={isCurrent ? 'animate-pulse' : ''}
            style={{
              width: 28,
              height: 28,
              backgroundColor: bgColor,
              border: `1px solid ${borderColor}`,
              color: textColor,
              display: 'flex',
              alignItems: 'center',
              justifyContent: 'center',
              fontSize: '5px',
              opacity: isLocked ? 0.5 : 1,
            }}
          >
            {isCompleted ? '\u2713' : dep.shortName}
          </div>
        );
      })}
    </div>
  );
}
```

The `title` attribute on each tile provides a hover tooltip with the full department name (no extra UI needed).

const unlockedRooms = useMemo(() => {
  // Try reading from pq:save:v2 unlockedDepartments if Phase 14 provides it
  try {
    const raw = localStorage.getItem('pq:save:v2');
    if (raw) {
      const save = JSON.parse(raw);
      if (Array.isArray(save?.unlockedDepartments)) return save.unlockedDepartments as string[];
    }
  } catch { /* ignore */ }
  // Fallback: reception is always unlocked; subsequent departments unlock when previous is complete
  const order = ['reception', 'break_room', 'lab', 'records', 'it', 'er'];
  const unlocked: string[] = ['reception'];
  for (let i = 0; i < order.length - 1; i++) {
    if (completedRooms.includes(order[i])) {
      unlocked.push(order[i + 1]);
    }
  }
  return unlocked;
}, [completedRooms]);
