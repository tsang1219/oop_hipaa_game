After ALL 8 textures exist (4 idle + 4 walk), register 4 walk animations at the END of `BootScene.create()`, AFTER `generatePlayerTexture()` and before `this.scene.start('HubWorld')`:

```typescript
const WALK_DIRS = ['down', 'up', 'left', 'right'] as const;
for (const dir of WALK_DIRS) {
  if (!this.anims.exists(`walk_${dir}`)) {
    this.anims.create({
      key: `walk_${dir}`,
      frames: [

      frameRate: 7,
      repeat: -1,

  }
}
```

```typescript
    this.lastFacingTexture = 'player_down';

// After the diagonal normalization block, add idle stop:
if (!left && !right && !up && !down) {
  this.player.anims.stop();
  this.player.setTexture(this.lastFacingTexture);
}
```

The `true` second argument to `play()` is `ignoreIfPlaying` — prevents animation restart jitter when holding a key. `lastFacingTexture` tracks the idle direction to restore on stop. Left/right priority on diagonal movement is preserved (existing behavior).
