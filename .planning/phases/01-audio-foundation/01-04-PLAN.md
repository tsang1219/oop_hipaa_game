Both pages already have:
```typescript
const gameRef = useRef<Phaser.Game | null>(null);
```

The Phaser.Game instance is accessible via gameRef.current after the game starts.
game.sound.setMute(true/false) silences/unsilences all sounds globally (all scenes).

Mute toggle pattern to add in each page:
```typescript
// 1. Add muted state (lazy initial reads localStorage):
const [muted, setMuted] = useState(() =>
  localStorage.getItem('sfx_muted') === 'true'
);

// 2. Add useEffect to apply to Phaser + persist:
useEffect(() => {
  if (gameRef.current?.sound) {
    gameRef.current.sound.setMute(muted);
  }
  localStorage.setItem('sfx_muted', String(muted));
}, [muted]);

// 3. Add button to HUD JSX:
<button
  onClick={() => setMuted(m => !m)}
  className="text-[10px] text-gray-300 hover:text-white transition-colors"
  title={muted ? 'Unmute' : 'Mute'}
>
  {muted ? '🔇' : '🔊'}
</button>
```

Add the mute button as a sibling of the CODEX button, at the end of this flex container.

PrivacyQuestPage HUD location (~line 433-442):
The HUD below the canvas `<div className="flex items-center gap-6">` contains KnowledgeTracker and ChecklistUI.
Add the mute button as a sibling inside this same flex container.

IMPORTANT: The muted useEffect has gameRef.current?.sound — use optional chaining because the game may not be initialized yet on first render. The effect fires whenever muted changes, and by then the game is running.

PITFALL: Do NOT use `game.sound.mute = true` (direct property). Always use `game.sound.setMute(bool)` — the setter properly notifies all active sound instances.
